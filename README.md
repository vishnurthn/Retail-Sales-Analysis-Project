# Retail Sales Analysis Project

## Overview

This project analyzes a real-world retail sales dataset to identify customer purchasing patterns, sales trends, and business insights using Python and data visualization techniques.

## Objectives

* Perform data exploration and cleaning
* Analyze customer demographics
* Identify sales trends and top-performing categories
* Visualize key business metrics
* Generate actionable business insights

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset Features

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Analysis Performed

1. Dataset Exploration
2. Missing Value Analysis
3. Sales by Product Category
4. Gender-wise Sales Analysis
5. Customer Age Distribution
6. Monthly Sales Trend Analysis
7. Correlation Analysis
8. Sales Distribution Analysis

## Key Findings

* The dataset contains 1000 retail transactions.
* No missing values were found.
* Certain product categories generated significantly higher revenue.
* Monthly sales showed variations throughout the year.
* Customer spending patterns varied across demographics.
* Quantity and Price per Unit strongly influenced Total Sales.
